# flashAction

封装闪烁动作

# Parameters

  • options object 选项
    • options.times number 闪烁次数，默认3次 (optional, default 3)
    • options.delay number 闪烁间隔时间(ms) (optional, default 100)
    • options.actionOdd function? 奇数回调
    • options.actionEven function? 偶数回调
    • options.recover function? 状态恢复回调

# Examples

// 文字闪烁，奇数次呈现为红色，偶数次成纤维蓝色，动画结束呈现为黑色
var text = $('#target span.txt');
$flashAction({
	actionOdd : function (){
		text.css('color', '#f00');
	},
	actionEven : function (){
		text.css('color', '#00f');
	},
	recover : function (){
		text.css('color', '#000');
	}
});

# Fx

  • See: https://mootools.net/core/docs/1.6.0/Fx/Fx

动画类 - 用于处理不适合使用 transition 的动画场景

可绑定的事件

  • start 动画开始时触发
  • complete 动画已完成
  • stop 动画尚未完成就被中断
  • cancel 动画被取消

# Parameters

  • options Object? 动画选项
    • options.fps Number 帧速率(f/s)，实际上动画运行的最高帧速率不会高于 requestAnimationFrame 提供的帧速率 (optional, default 60)
    • options.duration Number 动画持续时间(ms) (optional, default 500)
    • options.transition (String | Function)? 动画执行方式，参见 spore-kit-fx/transitions
    • options.frames Number? 从哪一帧开始执行
    • options.frameSkip Boolean 是否跳帧 (optional, default true)
    • options.link String 动画衔接方式，可选：['ignore', 'cancel'] (optional, default 'ignore')

# Examples

var fx = new Fx({
	duration : 1000
});
fx.set = function (now) {
	node.style.marginLeft = now + 'px';
};
fx.on('complete', function(){
	console.info('animation end');
});
fx.start(0, 600);  // 1秒内数字从0增加到600

# Fx#getTransition

设置动画的执行方式，配置缓动效果

# Examples

var fx = new Fx();
fx.getTransition = function () {
	return function (p) {
		return -(Math.cos(Math.PI * p) - 1) / 2;
	};
};

# smoothScrollTo

平滑滚动到某个元素，只进行垂直方向的滚动

  • requires jQuery/Zepto

# Parameters

  • node Object 目标DOM元素
  • spec Object 选项
    • spec.delta Number 目标滚动位置与目标元素顶部的间距，可以为负值 (optional, default 0)
    • spec.maxDelay Number 动画执行时间限制(ms)，动画执行超过此时间则直接停止，立刻滚动到目标位置 (optional, default 3000)

# Examples

// 滚动到页面顶端
smoothScrollTo(document.body);

# sticky

IOS sticky 效果 polyfill

  • requires jQuery/Zepto

# Parameters

  • node Object 目标DOM元素
  • options Object 选项
    • options.clone Boolean 是否创建一个 clone 元素 (optional, default true)
    • options.placeholder Object sticky 效果启动时的占位 dom 元素 (optional, default null)
    • options.disableAndroid Boolean 是否在 Android 下停用 sticky 效果 (optional, default false)
    • options.offsetParent Object 提供一个相对定位元素来匹配浮动时的定位样式 (optional, default null)
    • options.styles Object 进入 sticky 状态时的样式 (optional, default {})

# Examples

sticky($('h1').get(0));
