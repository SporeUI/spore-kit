# assign

扩展并覆盖对象

# Parameters

  • obj Object (opens new window) 要扩展的对象
  • item Object (opens new window) 要扩展的属性键值对

# Examples

var obj = {a: 1, b: 2};
console.info(assign(obj, {b: 3, c: 4})); // {a: 1, b: 3, c: 4}
console.info(assign({}, obj, {b: 3, c: 4})); // {a: 1, b: 3, c: 4}

# cover

覆盖对象，不添加新的键

# Parameters

  • object Object (opens new window) 要覆盖的对象
  • item Object (opens new window) 要覆盖的属性键值对

# Examples

var obj = {a: 1, b: 2};
console.info(cover(obj,{b: 3, c: 4}));	//{a: 1, b: 3}

# find

查找对象路径上的值

# Parameters

  • object Object (opens new window) 要查找的对象
  • path String (opens new window) 要查找的路径

# Examples

var obj = {a:{b:{c:1}}};
console.info(find(obj,'a.b.c')); // 1
console.info(find(obj,'a.c')); // undefined

# type

获取数据类型

# Parameters

  • item any 任何类型数据

# Examples

type({}); // 'object'
type(1); // 'number'
type(''); // 'string'
type(function(){}); // 'function'
type(); // 'undefined'
type(null); // 'null'
type(new Date()); // 'date'
type(/a/); // 'regexp'
type(Symbol('a')); // 'symbol'
type(window) // 'window'
type(document) // 'htmldocument'
type(document.body) // 'htmlbodyelement'
type(document.head) // 'htmlheadelement'
type(document.getElementsByTagName('div')) // 'htmlcollection'
type(document.getElementsByTagName('div')[0]) // 'htmldivelement'
