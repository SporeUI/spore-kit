# abToHex

ArrayBuffer转16进制字符串

# Parameters

  • buffer ArrayBuffer 需要转换的 ArrayBuffer

# Examples

var ab = new ArrayBuffer(2);
var dv = new DataView(ab);
dv.setUint8(0, 171);
dv.setUint8(1, 205);
abToHex(ab); // => 'abcd'

# hslToRgb

HSL颜色值转换为RGB

  • 换算公式改编自 http://en.wikipedia.org/wiki/HSL_color_space.
  • h, s, 和 l 设定在 [0, 1] 之间
  • 返回的 r, g, 和 b 在 [0, 255]之间

# Parameters

  • h Number 色相
  • s Number 饱和度
  • l Number 亮度

# Examples

hslToRgb(0, 0, 0); // => [0,0,0]
hslToRgb(0, 0, 1); // => [255,255,255]
hslToRgb(0.5555555555555555, 0.9374999999999999, 0.6862745098039216); // => [100,200,250]
1

Returns Array RGB色值数值

# job

任务分时执行

  • 一方面避免单次reflow流程执行太多js任务导致浏览器卡死
  • 另一方面单个任务的报错不会影响后续任务的执行

# Parameters

  • fn Function 任务函数

# Examples

job(function() {
	//task1 start
});
job(function() {
	//task2 start
});

# rgbToHsl

RGB 颜色值转换为 HSL.

  • 转换公式参考自 http://en.wikipedia.org/wiki/HSL_color_space.
  • r, g, 和 b 需要在 [0, 255] 范围内
  • 返回的 h, s, 和 l 在 [0, 1] 之间

# Parameters

  • r Number 红色色值
  • g Number 绿色色值
  • b Number 蓝色色值

# Examples

rgbToHsl(100, 200, 250); // => [0.5555555555555555,0.9374999999999999,0.6862745098039216]
rgbToHsl(0, 0, 0); // => [0,0,0]
rgbToHsl(255, 255, 255); // => [0,0,1]
