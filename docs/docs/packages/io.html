# Examples

// 统一引入 spore-kit
var $kit = require('spore-kit');
console.info($kit.io.getScript);

// 单独引入 spore-kit-io

# iframePost

封装 iframe post 模式

  • requires jQuery/Zepto

# Parameters

  • spec Object 请求选项
    • spec.form Object 要请求的表单 (optional, default null)
    • spec.url String 请求地址
    • spec.data (Array | Object) 请求数据
    • spec.target String 目标 iframe 名称 (optional, default '')
    • spec.method String 请求方式 (optional, default 'post')
    • spec.acceptCharset String 请求目标的编码 (optional, default '')
    • spec.jsonp String 传递给接口的回调参数名称 (optional, default 'callback')
    • spec.jsonpMethod String 传递给接口的回调参数的传递方式，可选['post', 'get'] (optional, default '')
    • spec.jsonpCallback String 传递给接口的回调函数名称，默认自动生成 (optional, default '')
    • spec.jsonpAddParent String 传递给接口的回调函数名称需要附带的前缀调用路径 (optional, default '')
    • spec.complete Function? 获得数据的回调函数
    • spec.success Function? 成功获得数据的回调函数

# Examples

document.domain = 'qq.com';
iframePost({
	url: 'http://a.qq.com/form',
	data: [{
		n1: 'v1',
		n2: 'v2'
	}],
	success: function (rs) {
		console.info(rs);
	}
});

# getScript

加载 script 文件

# Parameters

  • options Object 选项
    • options.src String script 地址
    • options.charset String script 编码 (optional, default 'utf-8')
    • options.onLoad Function? script 加载完成的回调函数

# Examples

getScript({
	src: 'https://code.jquery.com/jquery-3.3.1.min.js',
	onLoad: function () {
		console.info(window.jQuery);
	}
});
