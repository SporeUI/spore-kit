# comma

  • See: http://stackoverflow.com/questions/2901102/how-to-print-a-number-with-commas-as-thousands-separators-in-javascript (opens new window)

数字的千分位逗号分隔表示法

  • IE8 的 toLocalString 给出了小数点后2位: N.00

# Parameters

  • num Number (opens new window) 数字

# Examples

comma(1234567); //'1,234,567'
1

Returns String (opens new window) 千分位表示的数字

# fixTo

修正补位

# Parameters

  • num (Number (opens new window) | String (opens new window)) 要补位的数字
  • w Number (opens new window) w 补位数量 (optional, default 2)

# Examples

fixTo(0, 2);	//return '00'
1

Returns String (opens new window) 经过补位的字符串

# limit

限制数字在一个范围内

# Parameters

  • num Number (opens new window) 要限制的数字
  • min Number (opens new window) 最小边界数值
  • max Number (opens new window) 最大边界数值

# Examples

limit(1, 5, 10); // 5
limit(6, 5, 10); // 6
limit(11, 5, 10); // 10
1
2
3

Returns Number (opens new window) 经过限制的数值

# numerical

将数据类型转为整数数字，转换失败则返回一个默认值

# Parameters

  • str any 要转换的数据
  • def Number (opens new window) 转换失败时的默认值 (optional, default 0)
  • sys Number (opens new window) 进制 (optional, default 10)

# Examples

numerical('10x'); // 10
numerical('x10'); // 0
1
2

Returns Number (opens new window) 转换而得的整数
