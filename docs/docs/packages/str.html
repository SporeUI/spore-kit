# bLength

获取字符串长度，一个中文算2个字符

# Parameters

  • str String 要计算长度的字符串

# Examples

bLength('中文cc'); // 6
1

Returns Number 字符串长度

# dbcToSbc

全角字符转半角字符

# Parameters

  • str String 包含了全角字符的字符串

# Examples

dbcToSbc('ＳＡＡＳＤＦＳＡＤＦ'); // 'SAASDFSADF'
1

Returns String 经过转换的字符串

# decodeHTML

解码HTML，将实体字符转换为HTML字符

# Parameters

  • str String 含有实体字符的字符串

# Examples

decodeHTML('&amp;&lt;&gt;&quot;&#39;&#32;'); // '&<>"\' '
1

Returns String HTML字符串

# encodeHTML

编码HTML，将HTML字符转为实体字符

# Parameters

  • str String 含有HTML字符的字符串

# Examples

encodeHTML(`&<>"\' `); // '&amp;&lt;&gt;&quot;&#39;&#32;'
1

Returns String 经过转换的字符串

# getRnd36

获取36进制随机字符串

# Parameters

  • rnd Float? 随机数，不传则生成一个随机数

# Examples

getRnd36(0.5810766832590446); // 'kx2pozz9rgf'
1

Returns String 转成为36进制的字符串

# getTime36

获取36进制日期字符串

# Parameters

  • date Date? 符合规范的日期字符串或者数字，不传参数则使用当前客户端时间

# Examples

getTime36('2020'); // 'k4ujaio0'
1

Returns String 转成为36进制的字符串

# getUniqueKey

生成一个不与之前重复的随机字符串

# Examples

getUniqueKey(); // '166aae1fa9f'
1

Returns String 随机字符串

# substitute

简单模板函数

# Parameters

  • str String 要替换模板的字符串
  • obj Object 模板对应的数据对象
  • reg RegExp 解析模板的正则表达式 (optional, default /\\?\{\{([^{}]+)\}\}/g)

# Examples

substitute('{{city}}欢迎您', {city:'北京'}); // '北京欢迎您'
1

Returns String 替换了模板的字符串
